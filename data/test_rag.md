# RAG 技术简介

RAG (Retrieval Augmented Generation) 检索增强生成，是一种结合检索系统和生成模型的技术。

## RAG 的优势

1. **知识时效性**: 可以动态更新知识库，不需要重新训练模型
2. **可解释性**: 可以查看引用的来源
3. **成本效益**: 不需要昂贵的模型微调
4. **减少幻觉**: 生成内容基于真实文档

## RAG 工作流程

1. **文档加载**: 读取 PDF、Word、TXT 等格式的文档
2. **文本切分**: 将长文档分成小块
3. **向量化**: 将文本转为向量 embeddings
4. **存储**: 保存到向量数据库
5. **检索**: 根据用户问题检索相关文档
6. **生成**: 将检索结果提供给 LLM 生成回答

# Chroma 向量数据库

Chroma 是一个开源的向量数据库，专为 LLM 应用设计。

## 特点

- 轻量级，易于部署
- 支持多种 embedding 模型
- 可以与 LangChain 集成
- 支持持久化存储

# Ollama 本地大模型

Ollama 是一个让你在本地运行大语言模型的工具。

## 支持的模型

- Llama 3
- Mistral
- Qwen
- DeepSeek
- 等等

## 优势

- 无需 API Key
- 完全本地运行，数据不外泄
- 可以运行在个人电脑上
- 支持 GPU 加速
